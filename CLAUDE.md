- 毎回変更後に, bun run check && bun run lintを実行し、型エラーが存在しないことを確認しなさい。
- コメントは関数やメソッドのコメントドキュメントのみとし、インラインコメントは書かないで。
- undefinedの判定に if (!value) はやめて、 if(value === undefined) を使って。

- このプロジェクトにおけるterminology
    - ワールド座標
        - ゲーム内のオブジェクトの論理座標。オブジェクト間のインタラクション等に使用
    - キャンバス座標
        - キャンバスAPI上での座標。描画時に使用。キャンバスの左上が原点。
    - スクリーン座標
        - スクリーン上の座標。入出力処理やDOM APIとの連携に使用。ウィンドウの左上が原点。
